# Cassette images

Сайт подгружает фото кассет с Unsplash. Чтобы использовать свои изображения в стиле [tapedeck.org](https://tapedeck.org):

- Кассета **на белом фоне**, вид **сверху** (как на tapedeck).
- Рекомендуемый размер: **~800×600 px** (или пропорция 4:3).
- Положите файлы сюда и укажите пути в `src/lib/cassetteImages.ts` (например `/assets/cassette-type1.jpg`).

Типы: Type I (Ferro), Type II (Chrome), Type III (Ferrochrom), Type IV (Metal), Original.
