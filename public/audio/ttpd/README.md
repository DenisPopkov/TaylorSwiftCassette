# Треки TTPD для плеера

Положите сюда 5 файлов для сравнения по блокам кассет:

- `1.mp3` — Fortnight
- `2.mp3` — The Tortured Poets Department
- `3.mp3` — My Boy Only Breaks His Favorite Toys
- `4.mp3` — Down Bad
- `5.mp3` — So Long, London

Можно использовать одни и те же файлы или разные записи (например, с разных типов кассет).
